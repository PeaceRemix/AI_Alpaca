# AI_Alpaca
A cute alpaca ai agent&lt;3

## Installation
1.Clone this alpaca 
```bash
   git clone [https://github.com/PeaceRemix/AI_Alpaca.git]
```
2.Install the package
```bash
   pip install -r requirements.txt
```
## How to run
Open the terminal and enter
```bash
   streamlit run web_alpaca.py
```
(remember to use your own api key!)
